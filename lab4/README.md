## Getting Started

1. Make sure you have Docker and Docker Compose installed.
2. Run the database:

```bash
docker-compose up -d
```

3. To stop the database:

```bash
docker-compose down
```

## Connection Details

- Host: `localhost`
- Port: `5432`
- Username: `postgres`
- Password: `postgres`
- Database: `cryptography`

These values can be customized by editing the `.env` file.
